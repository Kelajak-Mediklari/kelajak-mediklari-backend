## Development Tools

### Create API Files

```bash
python utils/create_api_files.py app_name folder_name endpoint_name
```

Example: `python utils/create_api_files.py course course CategoryList` creates:
```
apps/course/api_endpoints/course/CategoryList/
├── __init__.py
├── doc.md
├── serializers.py
├── views.py
└── tests.py
```
